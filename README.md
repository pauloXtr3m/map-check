# Map-check app
Save annotations in the map

# Setup

### Environment
* Rename `.env.example` file to `.env`
* Set your email in env file

### Only on Android

Create a Google Maps key and add to

```
android/app/src/main/AndroidManifest.xml
```

### Install dependencies

```yarn```

### Only on iOS
```cd ios && pod install```

### Run iOS
In root folder:

```yarn ios```
### Run Android
In root folder:

```yarn android```
